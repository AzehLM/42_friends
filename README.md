# 42 Friends

A Chrome extension that adds a friend system to the 42 intra and displays monthly logtime below your weekly attendance.

## Installation

### Chrome

1. Download or clone this repository
2. Open Chrome and navigate to `chrome://extensions`
3. Enable **Developer mode** (toggle in top right)
4. Click **Load unpacked** and select the extension folder
5. Visit https://profile.intra.42.fr to use the extension


### Firefox

1. Install the extension from the [Firefox Marketplace](https://addons.mozilla.org/en-US/firefox/addon/42-friends/)
2. Visit https://profile.intra.42.fr to use the extension
